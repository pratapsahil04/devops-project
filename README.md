# 🚀 DevOps Pipeline Project

An end-to-end CI/CD pipeline for deploying a Java Spring Boot application using Git, GitHub, Maven, Jenkins, Docker, Kubernetes (Minikube), with future deployment to AWS.

---

## 📖 Project Overview

This project demonstrates a complete DevOps workflow from source code management to application deployment. Every stage is automated to simulate a real-world software delivery pipeline.

---

## 🛠 Tech Stack

| Category | Technology |
|----------|------------|
| Language | Java 17 |
| Framework | Spring Boot |
| Build Tool | Maven |
| Source Control | Git & GitHub |
| CI/CD | Jenkins |
| Containerization | Docker |
| Container Orchestration | Kubernetes (Minikube) |
| Cloud (Future) | AWS EC2, ECR, EKS |
| Infrastructure as Code | Terraform |
| Configuration Management | Ansible |
